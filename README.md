Instructions to Run Versions
-----------------------------------

Savor Server Ver: Needs to be run on Live Server on VSCode or hosted by another service like ngrok to be accessed.

savorApp (iOS Package Version): Needs to be run on XCode as it isn't published on an app store, only saved locally, and can be run on a simulation phone.

Savor_Android: (Android Package Version): Needs to be run in Android Studio or can be pushed to an Android phone to be run locally.
